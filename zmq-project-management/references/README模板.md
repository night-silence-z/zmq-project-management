# README 模板（用于生成 `00_管理/README.md`）

> 建档时按"模板"区创建；"当前状态"与"当前版指针表"每次收工必须更新——它们是所有 AI 会话开工重建上下文的第一读物。示例区仅供参照，创建时不要带入。

## 模板

```markdown
# {项目名} · 项目总览

## 启动卡

{按 references/启动卡模板.md 生成，签核后不再改动；变更走台账决策}

## 当前状态（每次收工更新）

- 阶段：{M1 / 二期 / …}
- 一句话进展：{当前干到哪了}
- 下一步：{最近的 1-3 件事}
- 阻塞：{无 / 等待什么}
- 最近更新：{YYYY-MM-DD HH:MM}

## 当前版指针表

| 产出物线 | 当前版文件 | 状态 | 对应关系 |
| -- | -- | -- | -- |
| {线名} | {路径/文件-vN} | 探索/候选/已定稿/已交付 | {如：对应 PRD v2} |

## 权威源清单

| 口径类型 | 权威文件 | 说明 |
| -- | -- | -- |
| {术语/指标/契约} | 00_管理/{口径字典.md 等} | 定义只出现在这里，产出只引用 |

## 目录说明

{仅在启用了扩展目录时列出，基础 4 目录不用写}
```

## 示例（指针表参照）

| 产出物线 | 当前版文件 | 状态 | 对应关系 |
| -- | -- | -- | -- |
| PRD | 01_产出/PRD/当前/落地页PRD-v2.md | 已定稿 | — |
| 原型 | 01_产出/原型/当前/落地页-v4.html | 候选——待用户验收 | 对应 PRD v2 |
| 汇报 | 01_产出/汇报/当前/阶段汇报-v1.html | 探索 | 取数口径见口径字典 v3 |

要点：一条线一行；"当前版文件"列只放纯路径，任何备注（如"待裁决"）写入"对应关系"列；状态只用阶梯词；跨线对应关系必须登记（原型改版时先查它对应的 PRD 是否还在当前版）。
